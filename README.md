#  Helix Jump — 3D Game (Unity)

Мобильная 3D-игра в жанре аркада, вдохновлённая классическим Helix Jump.  
Разработана полностью самостоятельно — от механик и физики до 3D-моделей.

---

##  Геймплей

Шарик падает вниз по спирали из платформ. Задача игрока — проваливаться сквозь просветы  
и избегать красных (опасных) секций. Чем ниже — тем сложнее.

---

## ️ Технологии

| Инструмент | Использование |
|---|---|
| Unity (C#) | Игровой движок, вся логика |
| Blender | Авторские 3D-модели (шар, платформы, ось) |
| Git | Контроль версий |

---

## ️ Структура проекта

```
Assets/
└── HelixJumpFS/
    ├── Animations/   # Анимации шара (прыжок, падение)
    ├── Materials/    # Материалы платформ (обычная / ловушка / финиш)
    ├── Models/       # 3D-модели (.fbx): шар, ось, сегмент, пол
    ├── Prefabs/      # Префабы: Ball, Floor, GUI, Background
    ├── Scenes/       # Игровые сцены
    ├── Scripts/      # Вся игровая логика на C#
    └── Sprites/      # UI-спрайты
```

---

##  Ключевые механики (реализованы с нуля)

- **Физика шара** — падение, отскок, управление касанием/мышью
- **Процедурная спираль** — платформы генерируются динамически
- **Система секций** — обычные / ловушки / финишная платформа
- **Анимации** — контроллер анимаций шара (прыжок при пробивании)
- **GUI** — игровой оверлей: счёт, уровень, экраны старта и проигрыша

---

##  Как запустить

1. Установить [Unity Hub](https://unity.com/download)
2. Клонировать репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИК/helix-jump.git
   ```
3. Открыть папку проекта через Unity Hub → **Open Project**
4. Открыть сцену `Assets/HelixJumpFS/Scenes/` и нажать **Play**
